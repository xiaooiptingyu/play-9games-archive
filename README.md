# play-9games-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存放，不针对任何特定域名或网站，仅作为 HTML 文件的集中存储与版本管理。

## 项目简介

`play-9games-archive` 是一个轻量级的 HTML 页面归档项目。仓库内包含一组独立的 HTML 文件，每个文件均可直接通过浏览器打开运行，不依赖外部服务器或数据库。适用于需要长期保存或共享的静态页面集合。

## 目录说明

```
play-9games-archive/
├── index.html          # 入口页面（可选）
├── pages/              # 主要页面存放目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── styles/
│   ├── scripts/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有归档的 HTML 页面，按需命名。
- `assets/`：存放页面引用的样式、脚本及图片资源。
- 根目录的 `index.html` 为可选入口，便于本地浏览。

## 页面归档说明

- 每个 HTML 页面均为独立完整文件，可离线运行。
- 页面之间无强制关联，可按需单独使用。
- 所有资源引用均采用相对路径，确保本地或静态托管环境下的兼容性。
- 不包含任何外部追踪、分析或广告代码。

## 维护说明

- 如需添加新页面，请将 HTML 文件放入 `pages/` 目录，并更新相关资源引用。
- 建议为每个页面添加简洁的注释，说明其用途或版本。
- 请勿在页面中包含指向具体域名或网站的链接，保持归档的中立性。
- 欢迎提交 Issue 或 Pull Request 进行内容增补或问题反馈。
